## Getting Started

Clone, install dependencies:

```bash
npm install
```
Run locally in dev mode:
```bash
npm run dev
```
Run locally in production mode:
```bash
npm run build
npm run start
```
Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
